# Student Management System (Java)

## 📌 Description
A simple Student Management System built using Java OOP concepts.  
This project allows users to manage student records easily through a console menu.

---

## 🚀 Features
- Add Student
- Display Students
- Search Student by ID
- Delete Student
- Update Student

---

## 🛠 Technologies Used
- Java
- Object-Oriented Programming (OOP)
- ArrayList
- Scanner

---
## 🎯 Purpose
This project was built for learning Java fundamentals and practicing OOP concepts.
